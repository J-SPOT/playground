## Getting Started

아래 모든 명령어는 frontend 루트디렉토리에서 실행

1. 백엔드 서버 실행
```bash
# 필요시 서버 빌드 후 실행
./server-build.sh

./server-start.sh
```
2. 프론트엔드 서버 실행
```bash
# node 패키지 설치 (초기 실행시 필수)
npm install

npm run dev
```

3. `localhost:3000` 접속
